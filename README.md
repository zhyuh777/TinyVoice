# TinyVoice

智能语音朗读播放器，将电子书转为自然语音，支持多格式导入、神经网络 TTS 合成、歌单管理。

## 功能

### 📖 电子书导入
- 支持 EPUB / TXT / PDF / MD 格式
- 自动章节识别与分段
- 对话与叙述自动分离

### 🎙 语音合成
- Microsoft Edge TTS 神经语音引擎
- 3 种女声 + 3 种男声，共 6 种音色
- 语速 0.5x - 3.0x 精调
- 拟真度调节（随机微扰动）
- 句间停顿自定义
- 情感强度控制
- 语气助词自然化处理

### 📻 歌单管理
- 默认「我的收藏」歌单
- 支持新建/删除歌单
- 导出音频自动加入歌单
- 支持导入本地 MP3/WAV 音频

### 🎵 播放器
- 歌单内曲目顺序播放
- 语速实时调节
- 音量控制
- 进度条拖动

### 🎨 界面
- Apple Music 风格设计
- 4 套主题：浅色 / 深色 / 多巴胺 / 缤纷
- 字体大小调节
- 界面缩放 (80%-150%)
- 书架 ↔ 歌单 Tab 切换
- 设置与导出弹窗

### ⚙ 设置
- 主题配色切换
- 默认导出路径自定义
- 所有设置自动持久化

## 安装

### macOS

下载 DMG 安装包，双击打开，将 TinyVoice 拖入 Applications 文件夹。

首次打开如提示「无法验证开发者」，右键点击 → 打开 → 确认。

### 开发运行

```bash
# 安装依赖
npm install

# 安装 Edge TTS（语音合成必需）
python3 -m venv .venv
source .venv/bin/activate
pip install edge-tts

# 启动
npm start
```

### 打包

```bash
npm run build:mac
```

## 技术栈

- Electron 33
- Edge TTS（微软神经语音）
- Node.js
- 原生 JavaScript（无框架）

## 使用说明

1. **导入电子书**：点击侧边栏「📖 导入电子书」，选择 EPUB/TXT/PDF/MD 文件
2. **导出音频**：点击书架中的书名 → 弹出导出设置 → 调整参数 → 开始导出
3. **播放**：切换到「📻 歌单」→ 点击曲目播放
4. **导入本地音频**：歌单页点击「📥 导入音频」
